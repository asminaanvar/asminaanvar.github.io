Project Details

Built a dashboard using Microsoft Excel, leveraging Pivot Tables, Slicers, Conditional Formatting, and Custom Charts to analyze call records, address key business questions, and deliver insights into sales trends, customer preferences, and representative performance.

Approach: Used Power Query to clean and transform raw call data, removing nulls and outliers. Applied Pivot Tables, charts, and slicers to analyze key metrics like call volume, revenue, duration, and satisfaction. Explored trends by day, gender, and region to uncover actionable insights.

Outcome: Helped managers identify performance gaps, monitor agent efficiency, and optimize revenue by highlighting peak time slots and regions. Suggested adding an extra agent in March and revealed that longer call duration doesn't directly influence satisfaction ratings.
